# Zig Invaders

A Space Invaders clone built with Zig, raylib, and flecs ECS.

## Dependencies

- [Zig](https://ziglang.org/) 0.16.0-dev (via nix)
- [raylib-zig](https://github.com/raylib-zig/raylib-zig)
- [zflecs](https://github.com/zig-gamedev/zflecs)

## Build & Run

```sh
nix develop
zig build run
```

## Controls

- Left/Right arrow — move player
- Space — shoot
